Color deficiency Simulation

  • Organization : Kats Laboratory of Applied Physics
  • Professor : Dr. Mikhail Kats
  • Duration : Ongoing (since December 2017)
  • Technologies : Matlab, Hyperspectral Imaging, Unity, Oculus
  • Description : We design algorithms to perform chromatic adjustment on hyperspectral images to simulate the vision of color vision deficiency (CVD). We then visualize our experiment results in virtual reality to simulate the experience
  • Application : This technique can be used to mathematically recreate the vision of Color Vision Deficiency and provide a realistic simulation through VR. It can also potentially be reverse engineered as a vision enhancement algorithm to allow humans more see more colors

Robotic Mimicry Control in VR

  • Organization : UW Graphics Group
  • Professor : Dr. Michael Gleicher
  • Duration : 1 year and 3 months
  • Technologies : Unity, ROS, Linux, StreamVR, IK Solver
  • Mimicry Control : In the first half of the researh, I implemented a virtual reality system that allows users to remote control robots with their hand and arm gestures by passing ROS (Robot Operating System) data between Unity and the robot through network socket with little latency. The video belows demonstrate a snippet of what it looks like.

  • Recreate Robot Movement : During the second half of the research, due to COVID-19 restriction, I was not able to be in the lab and research with the robot, so I switched gear into replicating movement by interpolating data from actual robotic experiments in Unity. By providing a spreadsheet of joint angles with timestamps, the interpretor I designed can create animation clips that replicate the exact robot arm movement. Here is a demo of what it looks like.
  • Application : This system allows us to perform robotic experiments without the actual robots, which can be beneficial to many fields such as the operation of dangerous equipments and more delicated robot motions, while reduce the cost of robot maintance

Neon Painter

  • Position : Developer
  • Duration : 8 months
  • Technologies : Unreal Engine, Blender
  • Description : Neon Painter is A 3D Painting VR application where user stands on top of a lone planet in randomly generated space scene. Combined with gorgeous graphics and intuitive painting system, user will have an immersive and expressive VR experience.
